Pyvene/nnsight Integration

MIT CSAIL · Jacob Andreas Lab · code ↗

Overview

Pyvene is an open-source toolkit for intervening on the internal activations of language models, a core tool for interpretability research. I built remote and batched execution for pyvene on NDIF's HPC GPU cluster, so that interventions can run against models hosted remotely rather than being limited to whatever fits on a local machine.

This unlocks large-scale intervention experiments across most open-weight models, letting researchers probe and steer model internals at a scale that was previously impractical. The work is being upstreamed in collaboration with Stanford NLP.

Highlights

  • Remote + batched execution for pyvene on NDIF's HPC GPU cluster.
  • Large-scale intervention experiments enabled across most open-weight models.
  • Upstreaming the integration with Stanford NLP.
