Scientific Knowledge Extraction

MIT CSAIL · Jacob Andreas Lab

Overview

This project extends the self-consistency framework to recover the latent scientific procedures that language models use when solving domain problems. Rather than just checking whether a model's answer is correct, the goal is to surface the underlying structure a model relies on: the implicit method behind its reasoning.

By exposing this model-discovered structure as auditable scientific claims, the work aims to turn a model's internal problem-solving procedures into something researchers can inspect, verify, and build on.

Highlights

  • Extends the consistency framework to recover latent scientific procedures in LMs.
  • Targets domain problem-solving, surfacing the implicit methods behind model reasoning.
  • Aims to present model-discovered structure as auditable scientific claims.
